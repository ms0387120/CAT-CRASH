# CAT-CRASH

* 語言：C
* 遊戲內容是藉由鍵盤操作(上下左右)來閃躲貓咪攻擊
* 過程中有背景音樂、雙角色設定、有趣的動畫供玩家欣賞，是一款很療癒的遊戲
* 遊戲難度依序為簡易、普通、困難、終極，不需玩家選擇難度，就讓玩家一次玩到底
